MIT6.00 Introduction to Computer Science and Programming 学习笔记（一）

发表于 2018-01-31 | 分类于 study

前言
在YouTube上第一次看到这个系列的视频纯属偶然，更准确地说，动机是必然的，具体内容却是未预期的。最初目的是为了覆盖自己主页上各种奇怪的推荐，不加选择地搜索各种技术性与理论向的视频，于是看到了这个公开课。这门课是MIT工科学生的大一课程，大部分内容属于一个计算机学生的必备知识，作为大四学生，内容十分基础。但我总有有种焦虑，认为名校的课程总归有值得学习的地方，即便是从头构建计算机的整体理论，我也应该把这个系列看完。

经过一段时间的学习（快进与跳过），不免惊叹于名校教师的讲课效率。一学期的课程，覆盖了在这边学习两年的内容，又不失内容的广度和深度，大概这也是我迫不及待地要把这些内容记录下来的一大动力。趁着博客上线的机会，我也终于决定把这个系列进行整理，列出对自己有启发性的内容，也算是没白费花这么多时间把首页reflesh过了。

写这篇推时，已经看到第19课，然而对之前内容的整理，依然得从头开始。要怪，只能怪自己没早点决定上线博客（笑）。尽管这个举动更多地是出于欠缺理论的焦虑，但毕竟也聊胜于无。如此，就尽快进入正题吧。

Hello World!

发表于 2018-01-30

删掉了原来自动生成的 hello world 文章，这个个人博客也算是正式开始使用了。

很惭愧的是，在大三上学期开始认识http原理之后萌生的开通个人博客的想法，直到现在才真正步入正轨。回想起来，当初装模作样地在自己的本地机上装了一套WAMP栈，还是一键打包的setup.exe那种，搞了几天也没有弄懂怎么发到网上去（现在我知道这个过程叫部署）。更搞笑的是，当时还在那个本地环境上发了一篇缅怀DNF的文章，洋洋洒洒近千字，内容当然是已经忘了，只记得在里面贴了一张自己很喜欢的职业，魔法师loli，的图片。庆幸的是，这张图片还找得到。

几经折腾，这个环境当然是已经被数次重装系统和换硬盘给折腾没了。如果有机会，也许我还会在这里重新构思那篇文章，这次肯定会注入更多的真情实感，而不是之前那样的矫揉造作。

我知道自己从来不是有恒心和耐力把一件事坚持很久的人，之前因为博客一直没上线，所以对自己的要求就变低了。现在我希望，在大家（虽然不一定有人看）的共同见证下，我能够持续更新这个博客，也算是真正步入钻研技术的道路。之所以这么说，大概是因为经过三年（实际上是两年）的专业知识学习之后，更加清楚地意识到了记录所知所想对学习的重要性，学而时习之，才能够温故而知新。对于我这样记忆功能缺陷的人来说，实在是太需要了。

所以，这个博客的主要定位还是一些技术和理论的总结以及反思。当然，像我这样的（自称的）思想的巨人，肯定还会有更多无病呻吟的随笔的。

如果你看到了这篇文章，就请你也祝福我能坚持更新下去吧。
